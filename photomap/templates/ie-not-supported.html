No Support for Internet Explorer

Making a website (web-application) using as much javascript and css as this one work with the Internet Explorer without major problems, is a lot of work. Therefor we decided to not support the IE for now. We might reconsider this decision once the app is working smoothly and we have enough time to work on something like IE-integration.

We're sorry for any inconvenience this might cause!

If you really want to use this app. You can download and install an alternative browser, which you use for this app. Here are a couple of links:
